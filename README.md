Awesome Chocolates Sales Dashboard is an interactive data analytics & visualization project that transforms raw chocolate sales data into insightful business intelligence using Power BI, MySQL, and advanced reporting techniques. This dashboard helps stakeholders quickly understand sales trends, performance across regions, shipment efficiency, and profitability.

---

## About This Project

This project takes a sales dataset for a fictional chocolate brand and builds an end-to-end analytics solution that provides visibility into key performance indicators:

- Total Sales & Revenue Trends
- Sales & Shipment Count by Region
- Profit Margin Analysis
- Top Performing Products
- Interactive Filters
- Drill-Down Capabilities

Visualizations were created using Power BI, with data shaping and modeling using MySQL / Power Query to build a dynamic, interactive representation of business performance. 

---

## Features

- **Interactive Power BI Dashboard**
  - Slicers for time period, product category, and location
  - Dynamic visuals including bar charts, cards, and maps
- **Data Processing**
  - Cleaned and transformed using Power Query
  - SQL scripts for extracting and aggregating key metrics
- **KPI Cards & Insights**
  - Total Revenue
  - Total Shipment Count
  - Profit Percentage
- **Sales by Geography**
  - Region-wise insights for better decision-making
